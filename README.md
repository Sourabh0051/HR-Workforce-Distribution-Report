# HR-Workforce-Distribution-Report

Project Overview

This project analyzes employee demographics, workforce distribution, and attrition trends using SQL and presents the insights through a Power BI dashboard. The objective is to transform raw HR data into clear, data-driven insights that support HR reporting and management decision-making.

Scope of Analysis:

Gender, age, and race distribution of active employees

Headquarters vs remote workforce comparison

State-wise employee distribution

Department-wise termination percentage

Workforce growth and employee count trends over time

Average length of employment

Project Deliverables:

Power BI Dashboard & PDF: Visual summary of workforce insights
SQL Analysis PDF: Contains all SQL queries used in the project, along with clear explanations and query outputs for transparency and reproducibility 

Tools & Technologies:

SQL – Data extraction, filtering, aggregations, and calculations
Power BI – KPI and interactive dashboard creation
PDF Reports – Structured documentation of insights and SQL logic

Outcome:

This project provides a comprehensive view of the organization’s workforce structure, highlighting key demographic patterns, location trends, and attrition insights. It demonstrates practical application of SQL and BI tools in real-world HR analytics.
